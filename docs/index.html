Methodology

Healthcare-specific scoring

Issues are scored for healthcare impact, not generic CVSS severity. A critical vulnerability on a ventilator scores higher than the same CVE on a back-office workstation.

Source authority tiers

Sources are weighted by authority. Tier 1 (CISA, FDA) carries more weight than Tier 4 (news). When sources disagree, the higher-tier source's assessment is noted.

AI usage

AI extracts facts, normalizes data, compares sources, and recommends from approved mitigation patterns. It does not invent mitigations or replace vendor guidance.

Evidence completeness

Each issue is scored 0–8 based on which fields are populated: title, summary, CVEs, severity, affected versions, vendor, handling warnings, and multi-source confirmation.

Playbook governance

All mitigation patterns cite specific standards (NIST SP 800-82, IEC 62443, FDA guidance, CISA best practices). Patterns are reviewed before inclusion. AI-generated recommendations are constrained to approved patterns only. See docs/playbook_governance.md for details.

About

AdvisoryOps is an open-source advisory-to-action pipeline for healthcare cybersecurity. It monitors 0 public sources, normalizes and deduplicates advisories, scores them for healthcare impact, and generates actionable defensive guidance.

The AI extracts, normalizes, compares, and recommends from approved mitigation patterns. It does not replace vendor guidance or make final operational decisions. All recommendations must be verified against vendor documentation before implementation in clinical environments.
